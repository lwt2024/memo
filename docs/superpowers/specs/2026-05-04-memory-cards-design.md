# 记忆卡片应用设计文档

## 概述

这是一个基于艾宾浩斯记忆法的在线记忆卡片应用，支持用户创建、管理和复习记忆卡片，并提供卡片组分享功能。

## 核心价值

- 帮助用户高效记忆知识
- 智能复习提醒，按照遗忘曲线推送复习
- 支持多种卡片类型（文字、富文本、图片、音频）
- 云端同步，多设备可用
- 社区分享，学习他人的卡片组

## 技术架构

### 前端技术栈
- React 18 + TypeScript
- Tailwind CSS（响应式设计）
- React Router（路由管理）
- Axios（API 请求）

### 后端技术栈
- Node.js + Express + TypeScript
- PostgreSQL（关系型数据库）
- Redis（缓存、Session、推送队列）

## 数据模型

### users（用户表）
- id: 用户ID
- email: 邮箱
- password_hash: 密码哈希
- nickname: 昵称
- avatar: 头像URL
- created_at: 创建时间

### decks（卡片组表）
- id: 卡片组ID
- user_id: 创建者用户ID
- name: 卡片组名称
- description: 描述
- is_public: 是否公开
- created_at: 创建时间

### cards（卡片表）
- id: 卡片ID
- deck_id: 所属卡片组ID
- front: 正面内容
- back: 背面内容
- card_type: 卡片类型（text/rich_text/image/audio）
- media_urls: 媒体文件URL数组
- created_at: 创建时间

### review_records（复习记录表）
- id: 记录ID
- card_id: 卡片ID
- user_id: 用户ID
- ease_level: 难易程度（1-5）
- next_review_at: 下次复习时间
- last_review_at: 上次复习时间
- review_count: 复习次数

### deck_imports（导入记录表）
- id: 记录ID
- deck_id: 卡片组ID
- user_id: 导入用户ID
- imported_at: 导入时间

## 艾宾浩斯复习算法

### 基础间隔
遗忘曲线间隔：1天 → 2天 → 4天 → 7天 → 15天 → 30天...

### 动态调整
根据用户选择的难易程度（ease_level）调整间隔：
- 1（忘记）：间隔缩短，重新学习
- 2（困难）：间隔缩短 30%
- 3（一般）：按基础间隔
- 4（简单）：间隔延长 30%
- 5（太简单）：间隔延长 50%

### 复习流程
1. 显示卡片正面
2. 用户思考答案
3. 翻转显示卡片背面
4. 用户选择记忆程度
5. 计算下次复习时间并保存

## 功能模块

### Phase 1：核心功能
1. 用户认证模块
   - 注册/登录（邮箱 + 密码）
   - JWT Token 认证
   - 个人资料管理

2. 卡片管理模块
   - 创建/编辑/删除卡片组
   - 创建/编辑/删除卡片
   - 支持四种卡片类型
   - 卡片组内搜索和排序

3. 复习模块
   - 艾宾浩斯算法实现
   - 复习界面（翻转卡片、难度选择）
   - 今日待复习统计

4. 应用内提醒
   - Dashboard 统计展示
   - 待复习卡片提示

### Phase 2：分享功能
1. 分享模块
   - 公开分享（卡片组出现在社区广场）
   - 私密分享（生成分享链接/邀请码）
   - 导入功能（复制到自己账户）

2. 社区广场
   - 浏览公开卡片组
   - 分类/热度/最新筛选
   - 搜索功能

### Phase 3：增强功能
1. 浏览器推送通知
   - Service Worker + Web Push API
   - 用户可开启/关闭推送
   - 复习时间到推送提醒

2. 社区互动
   - 点赞卡片组
   - 收藏卡片组
   - 评论功能

## 页面结构

### 路由
- `/` - 首页 Dashboard（登录后）
- `/login` - 登录页
- `/register` - 注册页
- `/decks` - 我的卡片组列表
- `/decks/:id` - 卡片组详情
- `/decks/:id/review` - 复习界面
- `/community` - 社区广场
- `/settings` - 个人设置

## 响应式设计

### 电脑端（≥1024px）
- 侧边导航栏（固定）
- 主内容区域最大宽度 1200px
- 卡片列表：3-4列网格

### 平板端（768px - 1023px）
- 顶部导航栏 + 汉堡菜单
- 卡片列表：2列网格
- 复习界面：卡片宽度 80%

### 手机端（<768px）
- 底部导航栏
- 卡片列表：单列
- 复习界面：全屏卡片，滑动切换
- 底部固定操作按钮
